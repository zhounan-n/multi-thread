二叉树：
    每个节点最多有两个叶子结点
完全二叉树：
    若设二叉树的深度为h，除第 h 层外，其它各层 (1～h-1) 的结点数都达到最大个数，第 h 层所有的结点都连续集中在最左边，这就是完全二叉树。
平衡二叉树：
    它是一颗空树或者左右两个子树的高度不超过1，并且左右两颗子树都是平衡二叉树
    (实现有avl树 红黑树 替罪羊树、Treap、伸展树等)
二叉查找树：
    对于树中的每个节点X，它的左子树中的所有项的值小于X中的项，而它的右子树种的所有值大于X中的项

AVL树：
    带有平衡条件的二叉查找树
